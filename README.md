# Cours Git & Docker

Bienvenue dans les cours Git et Docker ! Ce dossier contient deux cours pratiques pour apprendre les outils essentiels du développement moderne.

## Cours disponibles

### [Cours Git](./cours/GIT-COURS.md)
Apprends à maîtriser Git, l'outil de versioning incontournable :
- Configuration et premiers pas
- Gestion des branches et fusion
- Résolution de conflits
- Collaboration et remote repositories
- Défis avancés pour aller plus loin

### [Cours Docker](./cours/DOCKER-COURS.md)
Découvre Docker et la containerisation :
- Concepts fondamentaux des containers
- Exercices pratiques avec Node.js, Alpine Linux et PHP
- Commandes Docker essentielles
- Cycle de vie des containers

### [Cours Docker Avancé](./cours/DOCKER-COURS-2.md)
Maîtrise Docker en profondeur :
- Création de Dockerfiles optimisés
- Docker Compose pour les applications multi-services
- Intégration CI/CD avec GitHub Actions
- Bonnes pratiques de production

### Comment utiliser ces exercices

1. **Lis d'abord** l'objectif de l'exercice
2. **Essaie par toi-même** avant de regarder les spoilers
3. **Utilise les spoilers** si tu es bloqué
4. **Expérimente** au-delà des instructions
5. **Pose-toi les questions** à la fin de chaque exercice

### Conseils

- Les **spoilers** contiennent les commandes exactes
- N'hésite pas à **expérimenter** et faire des erreurs
- **Relis** les concepts théoriques si nécessaire
- **Pratique régulièrement** pour bien mémoriser
